# Contributing to Universal Debian Persistence USB

We love contributions! Here’s how you can help:

1.  **Report Bugs:** If the installer fails on your specific USB or hardware, please open an Issue.
2.  **Add Flavors:** Want to add support for Ubuntu, Kali, or Arch? Submit a Pull Request!
3.  **Improve the Script:** If you're a Bash wizard and see a way to make the script safer or faster, let us know.

## Future Goals
- Support for multiple distros.
- GUI version of the installer.
- Auto-updating persistence partitions.
